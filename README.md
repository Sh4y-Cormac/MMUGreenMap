# MMUGreenMap
What is MMUGreenMap? MMUGreenMap is a web application that allows Multimedia University students to view
<br>
key accessibility and useful facilities around them by highlighting their location on MMU's map (via Google Maps).
<br>
This web app was made as part of the course LSSW6112 Sustainable Society by Group S.

# Stack and tools involved
1. Firebase for hosting and database
2. HTML, CSS and JavaScript for the web app interface
3. Google JavaScript Map API
